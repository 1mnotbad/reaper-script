Installation

Setup

There are two ways to install imnotbad-scripts: via ReaPack (recommended) or manually.

Method 1 — ReaPack (Recommended)

ReaPack is a package manager for REAPER. Download it from reapack.com.

  1. Open REAPER → Extensions → ReaPack → Import repositories…
  2. Paste the URL below and click OK:
https://1mnotbad.github.io/reaper-script/
  1. Go to Extensions → ReaPack → Browse packages…
  2. Search for imnotbad, select scripts, click Install/update selection.
  3. Restart REAPER when prompted.
💡 Tip

The shared imnotbad_lib.lua library is installed automatically as a dependency.

Method 2 — Manual

  1. Clone the repository:
git clone https://github.com/1mnotbad
  1. Copy files to your REAPER Scripts folder:
# Windows
%APPDATA%\REAPER\Scripts\imnotbad-scripts\

# macOS
~/Library/Application Support/REAPER/Scripts/imnotbad-scripts/

# Linux
~/.config/REAPER/Scripts/imnotbad-scripts/
⚠ Warning

Make sure imnotbad_lib.lua is in the same folder as the other scripts.

Verifying the Install

local lib = dofile(reaper.GetResourcePath() .. "/Scripts/imnotbad-scripts/imnotbad_lib.lua")
reaper.ShowConsoleMsg("version: " .. lib.version .. "\n")

Expected: version: 2.1.0

Quick Start

Beginner

Run your first script and set up a keyboard shortcut in under 5 minutes.

1. Run a Script

Open the REAPER Action List (Shift+/), click Load…, navigate to your scripts folder, and double-click any script to run it.

2. Assign a Keyboard Shortcut

In the Action List, select the action and click Add…. Press your desired key combination and click OK.

💡 Tip

Good choices: Ctrl+Shift+Letter or F9–F12.

3. Add to Toolbar

Right-click the toolbar → Customize toolbar… → find your script → click Add.

4. Your First Custom Script

-- my_script.lua
local script_path = debug.getinfo(1, "S").source:match("@?(.*[/\\])")
local lib = dofile(script_path .. "imnotbad_lib.lua")

local tracks = lib.tracks.getSelected()
for _, t in ipairs(tracks) do
  reaper.ShowConsoleMsg(lib.tracks.getName(t) .. "\n")
end

Routing Utils

Scriptimnotbad_mixer_routing.lua

Bulk-editing track routing. Works on selection or all tracks.

Actions

Action nameDescription
imnotbad: Route selected to parentRoutes all selected tracks to their parent folder track.
imnotbad: Disconnect all sendsRemoves every send from selected tracks.
imnotbad: Audit routing (console)Prints a routing diagram to the REAPER console.

Example

reaper.Undo_BeginBlock()
for _, track in ipairs(lib.tracks.getSelected()) do
  local parent = reaper.GetParentTrack(track)
  if parent then lib.tracks.setSend(track, parent, true) end
end
reaper.Undo_EndBlock("Route selected to parent", -1)

MIDI Humanizer

Scriptimnotbad_midi_humanize.lua

Adds controlled randomness to MIDI notes — velocity, position, and note length.

Parameters

ParameterTypeDescription
vel_rangeinteger±velocity variance (0–64). Default: 12
pos_rangenumber±position in beats (0–0.5). Default: 0.01
len_rangenumber±length in beats (0–0.25). Default: 0.0

Example

lib.midi.humanize({ vel_range = 15, pos_range = 0.008 })

Home › Scripts › FX Chain

FX Chain

Script imnotbad_fx_chain.lua new in v2.1

Save and restore entire FX chains (including plugin states) to named slots.

Usage

  • Save chain: Select a track → imnotbad: Save FX chain… → enter a name.
  • Restore chain: Select a track → imnotbad: Load FX chain… → pick a slot.
⚠ Note

Chains are saved as .rfxchain files in your REAPER FXChains folder, compatible with the built-in chain browser.
